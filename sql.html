Week 7 - SQL

The week consisted of SQL, the power and importance of data. Data processing was discussed, cleaning of data, what relational databases are, how to create databases, what the relationship between primary keys, foreign keys are. To me the most valuable part was how to incorporate SQL into Python.

The following problems where included:

  • Songs - Write SQL queries to answer questions about a database of songs.
  • Movies - Write SQL queries to answer questions about a database of movies.
  • Fiftyville - Write SQL queries to solve a mystery.
All three problems were interesting and provided their own challenges. I include some queries from Fiftyville as I am proud of how I used the Intersect operator.

The query returns only the common data in all three queries.

											
SELECT name
  FROM people
    JOIN bakery_security_logs
    ON people.license_plate = bakery_security_logs.license_plate
  WHERE people.license_plate = bakery_security_logs.license_plate
    AND year = 2021
    AND month = 7
    AND day = 28
    AND hour = 10
    AND minute BETWEEN 15 AND 25
INTERSECT
SELECT name
  FROM people
    JOIN bank_accounts
    ON people.id = bank_accounts.person_id
    JOIN atm_transactions
    ON bank_accounts.account_number = atm_transactions.account_number
  WHERE year = 2021
    AND month = 7
    AND day = 28
    AND atm_location = "Leggett Street"
    AND transaction_type = "withdraw"
INTERSECT
SELECT name
  FROM people
    JOIN phone_calls
    ON people.phone_number = phone_calls.caller
  WHERE year = 2021
    AND month = 7
    AND day = 28;
