C Language

Starting off strong, I was introduced to the language C. Powerful in that you control every aspect of memory management and creation of functions.

Covering week 1 through 5, and only including some code from the problem sets done. this is to honour the CS50 Academic Honesty


I was taught three aspects when evaluating code... correctness, design & style. The week covered IDEs, compilers, interfaces and how various code editors function. Functions, arguments, return values, variables, conditionals and boolean expressions were introduced along with some syntactic sugar. The challenge was to sort the pairs that would display the winner of any given pair


// Sort pairs in decreasing order by strength of victory
void sort_pairs(void)
{
	// Loop through the Pairs array and sort in descending order. Pair count used as the length as it holds the total number of pairs available.
	// https://www.javatpoint.com/c-program-to-sort-the-elements-of-an-array-in-ascending-order -- resource used to check selection sort.
	// The trick is to sort the pairs, which are arrays in themselves.
	// First start with a variable to hold the index that is being looped through and a variable to check what the difference is in the victory of the first pair at index i
	for (int i = 0; i < pair_count; i++)
	{
		int index = i;
		int strength = preferences[pairs[i].winner][pairs[i].loser] - preferences[pairs[i].loser][pairs[i].winner];
		// Loop through the pairs at i + 1 (the next pair) and assign it to a temp variable to compare to the initial pair. This loop checks which pair would be the max value
		for (int j = i + 1; j < pair_count; j++)
		{
			int temp_strength = preferences[pairs[j].winner][pairs[j].loser] - preferences[pairs[j].loser][pairs[j].winner];
			// if the strength of victory on the temp pair is more than the strength value, the index changes to value j (the next pair to check) and strength variable is calculated again
			if (temp_strength > strength)
			{
				index = j;
				strength = preferences[pairs[j].winner][pairs[j].loser] - preferences[pairs[j].loser][pairs[j].winner];
			}
		}
		// The pairs array is assigned to have the highest strength of victory and swapped with the lower value so the loop starts at the next index and the highest value is in the correct place.
		pair temp = pairs[index];
		pairs[index] = pairs[i];
		pairs[i] = temp;
	}
	return;
}

// Hashes word to a number
unsigned int hash(const char *word)
{
	// I used the following resources to learn more about hash tables and has functions
	// https://www3.cs.stonybrook.edu/~skiena/214/lectures/lect21/lect21.html
	// https://youtu.be/wg8hZxMRwcw
	// https://youtu.be/2Ti5yvumFTU
	// http://www.openbookproject.net/books/pythonds/SortSearch/Hashing.html

	// Variable to hold the total value of the word, adding each letter ASCII value together
	unsigned long total = 0;
	// Loop through each character of the word and adding the lower case ASCII value of the word to the total
	for (int i = 0, len = strlen(word); i < len; i++)
	{
		total += tolower(word[i]);
	}
	// Calculate the hash table index, total modulus number of buckets
	total = total % N;
	return total;
}
